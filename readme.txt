== Description ==

Satisfy is a rich, responsive and customizable WordPress theme. It's built using html5 and bootstrap with a modern design that's both desktop and mobile friendly, and supports all modern browsers and even older ie versions. The themes focus is mainly for blogs but itâ€™s also ideal for static, business and other types of sites and is quite easy to set up. It includes 2 new widgets and 3 page templates and when it comes to customization options Satisfy offers plenty. Color options, fonts, icons, featured images, full width headers and much more!

== Installation ==

1. Go to Appearance > Themes in your admin panel and hit Add New
2. Search for Satisfy and hit Install
3. When installed hit Activate to start using the theme
4. Navigate to Appearance > Satisfy info in your admin panel for more info how to set it up (optional)

== Widgets included ==

* Satisfy Latest Posts - Latest posts with thumbnail images (uses posts featured images)
* Satisfy Image - Displays an image
